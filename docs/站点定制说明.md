# 站点定制说明（给网站管理者）

更换 **logo / 标题 / 描述** 只需改**一个文件**：`src/config/site.ts`，然后重新构建即可。

---

## 一、改哪里

打开 `src/config/site.ts`：

```ts
export const siteConfig = {
  name: "字幕翻译",            // ① 左上角名称 + 浏览器标签页
  title: "字幕翻译",            // ② 浏览器标签页标题（留空则用 name）
  description: "上传字幕…",     // ③ SEO/分享卡片描述（meta description）
  logoSrc: "",                  // ④ logo 图片地址（见下），留空则用文字徽标
  logoText: "字",               // ⑤ 文字徽标：logoSrc 为空时显示的字
};
```

| 想改什么 | 改哪一项 |
|---|---|
| 网站名称（左上角、标签页） | `name` |
| 浏览器标签页标题 | `title`（留空自动用 `name`） |
| 站点描述（搜索引擎/分享） | `description` |
| 用**图片**当 logo | `logoSrc`（见第二节） |
| 用**文字**当 logo | `logoText`（保持 `logoSrc` 为空） |

---

## 二、换成图片 logo

1. 把图片（建议 SVG 或透明 PNG，正方形）放进项目根目录的 **`public/`** 文件夹，例如 `public/logo.svg`。
2. 在 `site.ts` 里把 `logoSrc` 填成 **以 `/` 开头的路径**：

   ```ts
   logoSrc: "/logo.svg",
   ```
3. logo 会显示在左上角（约 28×28 像素，`object-contain` 自适应）。想换回文字徽标，把 `logoSrc` 改回 `""` 即可。

> 浏览器标签页图标（favicon）：把 `favicon.ico` 放进 `public/` 即自动生效。

---

## 三、改完怎么生效

定制属于**构建期固化**，改完必须重新构建：

- 本地预览：`npm run dev`，刷新浏览器即可看到。
- 部署上线：重新构建后再部署。
  - 用宝塔「Node 项目」(`npm start`)：在服务器 `npm run build` 后「重启」。
  - 用发行版包：本机 `npm run release` 重新打包后上传（见 [发布流程](发布流程.md)）。

---

## 四、常见问题

**Q：改了没生效？**
A：确认是否重新构建/重启；浏览器强刷（Ctrl+F5）清缓存。

**Q：logo 图片不显示？**
A：确认图片在 `public/` 下、`logoSrc` 以 `/` 开头且文件名大小写一致；构建后再看。

**Q：能改主题色/按钮颜色吗？**
A：那属于样式层（Tailwind / `globals.css`），不在本说明范围；需开发改 `src/app/globals.css` 的共享控件类。
